# Antd-Beautify Plugin

HydroOJ 插件，为系统添加现代化的 Ant Design 风格美化效果。

## 功能特性

### 题目详情页卡片美化
- 将题目信息移至右侧边栏独立卡片中
- **题目标题**：包含收藏按钮和题目标题（如 `#D0017. 数字游戏`）
- **统计信息**：难度、通过率、尝试次数、已通过数量
  - 难度：紫色渐变
  - 通过率：绿色渐变
  - 尝试次数：蓝色渐变（可点击跳转到提交记录）
  - 已通过：青色渐变（可点击跳转到通过记录）
- **分数显示**：当查看某个提交记录时，显示当前得分
- **详细信息**：ID、类型、时间限制、内存限制、上传者
- **标签区域**：显示在卡片底部，点击可跳转到对应分类
- 支持浅色/深色模式自动切换

### 搜索框美化
- 将搜索框移至侧边栏独立卡片中
- 现代化的圆角输入框设计
- 蓝色主题搜索按钮
- 支持浅色/深色模式自动切换

### 标签分类着色
- **动态规划系列**（紫色）：动态规划、LCS、LIS、背包、DP、区间动归
- **贪心/树结构系列**（绿色）：贪心、构造、打表、树结构、LCA、生成树
- **图结构系列**（蓝色）：图结构、图论、最短路、拓扑排序、二分图匹配
- **数据结构系列**（青色）：数据结构、栈、队列、线段树、树状数组
- **数学系列**（红色）：数学与数论、快速幂、排列组合、矩阵、前缀和差分
- **搜索算法系列**（橙色）：搜索算法、DFS、BFS、递推、递归
- **分治/字符串系列**（粉色）：分治、字符串、KMP、后缀数组、一维/二维数组
- **基础算法技巧**（天蓝色）：二分查找、三分查找、双指针扫描、排序
- **函数问题**（黄色）：函数问题

### 按钮美化
- 现代化的蓝白风格按钮
- 悬停和点击动画效果

### 表格样式优化
- 深色模式下表头样式优化
- 代码框美化

### 全局过渡动画
- 平滑的页面过渡效果

## 安装方式

1. 将本目录放置于 `packages/ui-default/addon/` 目录下
2. 在 HydroOJ 管理后台启用插件
3. 重启 HydroOJ 服务

## 使用说明

插件启用后自动生效，无需额外配置。

## 兼容性

- HydroOJ >= 2.0
- 支持现代浏览器（Chrome, Firefox, Safari, Edge）
